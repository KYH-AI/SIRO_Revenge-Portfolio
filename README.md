## 📝 소개

[시로의 복수] 게임은 [2D] 장르의 [플랫폼(PC)]용 게임으로, [대학교 팀 프로젝트 과제]로 [3개월] 개발기간 동안 [Unity] 엔진을 이용해 진행되었습니다.
